# Enumeration: EventStreamMode

EventStreamMode は、EventStream のイベント受信モードを表します。

## Enumeration members

### AllEvent

• **AllEvent** = `0`

___

### RoomEvent

• **RoomEvent** = `1`

デバイスとデバイスが入室するルームのイベントのみ受信。
